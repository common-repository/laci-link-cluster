== Description ==

The Link and Cluster plugin allows you to easily manage internal links within your site, improving SEO and user experience. With this plugin, you can automatically convert specified keywords into internal links, helping to boost your site's visibility and navigation.

== FEATURES ==

- Simple and intuitive interface
- Customizable link options
- Support for hierarchical link structures
- Automatic link validation
- SEO-friendly URLs
- Compatibility with popular themes and plugins

== Installation ==

1. Download the plugin from wordpress.org, unpack it and upload the **[linkandcluster]** folder to your **wp-content/plugins/** directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings page and configure the desired options.

== Frequently Asked Questions ==

= What is internal linking and why is it important? =

An internal link connects one page to another within the same website domain. Internal links play a crucial role in SEO and contribute significantly to the overall optimization of a site.

= Will this plugin affect my site's speed? =

Not at all. The Link and Cluster is specifically built for speed and optimal performance. It creates links using a dedicated index, ensuring no extra load time on the frontend.
